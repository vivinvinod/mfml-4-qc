MFML-4-QC is an open-source library that enables multifidelity machine learning for quantum chemical systems. While the multifidelity methods are model architecture agnostic, this library provides a lightweight kernel ridge regression setup as the primary architecture.

🪲 **Bug Reports**: If you find a bug in MFML-4-QC, or have a feature request, please open a GitHub issue.
